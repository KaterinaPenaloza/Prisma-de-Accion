# Prisma de acción 💎

Aplicación de escritorio para organizar tareas usando la Matriz de Eisenhower.


## Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar aplicación
npm start
```

## Atajos de Teclado

- `Ctrl/Cmd + 1-4`: Agregar tarea rápida en cuadrante 1-4
- `Enter`: Guardar tarea en el modal
- `Esc`: Cerrar modal

## Cuadrantes

-  🔥 Q1: Urgente e Importante

-  📅 Q2: No Urgente pero Importante  

-  ⚡ Q3: Urgente pero No Importante

-  🗑️ Q4: No Urgente ni Importante


## 🛠️ Tecnologías

- Electron
- HTML/CSS/JavaScript vanilla
- LocalStorage para guardado de datos
